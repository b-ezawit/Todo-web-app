# Todo List Web Application

A simple and interactive **Todo List Web App** built using **HTML, CSS, and JavaScript**.  
Manage your daily tasks efficiently with real-time updates and a clean interface.

## Features
- **Add Tasks:** Quickly add new tasks to your list.
- **Delete Tasks:** Remove tasks you’ve completed or no longer need.
- **Mark as Completed:** Keep track of finished tasks with a simple toggle.
- **Responsive UI:** Works on desktops, tablets, and mobile devices.
- **Real-Time DOM Updates:** All actions are instantly reflected on the page.

## Technologies Used
- **HTML** – Structure of the application
- **CSS** – Styling and responsive design
- **JavaScript** – Dynamic task management and DOM manipulation

## Key Learnings
- Handling user input and events with JavaScript
- Manipulating the DOM dynamically
- Implementing basic CRUD functionality (Create, Read, Update, Delete)
- Designing a clean and user-friendly interface
